Evaluación Bootstrap

Fecha y hora límite de entrega: Viernes 9 de setiembre de 2017, 5:30 p.m.

Descripción

Esta evaluación consiste en continuar la página web desarrollada en la evaluación anterior, incorporando la biblioteca Bootstrap estudiada en clase para lograr una diagramación similar a la planteada en el wireframe proporcionado. Además, debe usar los elementos básicos proporcionados por Bootstrap (por ejemplo los páneles) para mejorar la apariencia y usabilidad de la página.

Debe entregarse un archivo comprimido en formato ZIP que incluya el archivo HTML, la biblioteca bootstrap y cualquier imagen enlazada desde la página. Todas las imágenes deben utilizar enlaces relativos y deben poder visualizarse sin problemas a la hora de abrir la página en el navegador directamente (sin necesidad de un servidor web).

Los criterios que se tomarán en cuenta para la evaluación son los siguientes:

  • Implementación correcta del grid para el layout del sitio (60%)
  • Uso de panels y otros elementos de bootstrap para los componentes del sitio (40%)
