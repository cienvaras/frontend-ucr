Evaluación CSS

Fecha y hora límite de entrega: Jueves 14 de setiembre de 2017, 5:30 p.m.

Descripción

Esta evaluación consiste en continuar la página web desarrollada en la evaluación anterior, incorporando los elementos de CSS estudiados en clase, para:

  1. Lograr una diagramación similar a la planteada en el wireframe proporcionado.
  2. Mejorar la apariencia visual predeterminada definida por el navegador

Debe entregarse un archivo comprimido en formato ZIP que incluya el archivo HTML, la(s) hoja(s) de estilos CSS utilizada y cualquier imagen enlazada desde la página. Todas las imágenes deben cargarse localmente y utilizar enlaces relativos y deben poder visualizarse sin problemas a la hora de abrir la página en el navegador directamente (sin necesidad de un servidor web).

Los criterios que se tomarán en cuenta para la evaluación son los siguientes:

  • Presencia en la página implementada de todos los componentes del wireframe (10%)
  • Implementación correcta del grid para el layout general del sitio usando float (30%)
  • Layout correcto de otros elementos de la página (Ej. enlaces de redes sociales, sección "Artículos Relacionados"). Debe utilizar Flexbox al menos en uno de los componentes (15%)
  • Uso de paddings y márgenes para controlar el espaciado entre los elementos (15%)
  • Uso correcto de una tipografía de Google Fonts (10%)
  • Cambio de los colores predeterminados de texto y fondo donde corresponda (10%)
  • Cambio del tamaño predeterminado del tamaño del texto usando unidades rem (10%)
